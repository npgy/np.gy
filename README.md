# np.gy
This is my personal website, accessible at (https://np.gy)

It is statically generated using [11ty](https://github.com/11ty/eleventy/) with the templating language [Nunjucks](https://github.com/mozilla/nunjucks).
I've also opted to use the awesome CSS framework of [Tailwind](https://github.com/tailwindlabs/tailwindcss).

This repository only contains the build files.
When compiled, the website only takes up 7.2kB exclusing my resume PDF.
